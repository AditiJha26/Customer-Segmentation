# Customer Segmentation

Segmented 401,604 customer transaction records using PCA for dimensionality reduction, reducing feature space by 40% while retaining 90% of the variance.
Compared K-Means and GMM clustering, achieving a silhouette score of 0.87 for K-Means (vs. 0.75 GMM), demonstrating superior cluster separation and interpretability.
Developed actionable customer segmentation insights, revealing distinct buyer personas, leading to a potential 25% increase in targeted marketing efficiency
